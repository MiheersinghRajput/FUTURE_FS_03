# 🏋️ Fitness Gym Website

This project is a professional business website developed as part of my Full Stack Web Development Internship (Task 3).

## 📌 Project Overview
The website is designed for a local fitness gym to showcase its services, pricing plans, and contact information. It provides a modern and responsive interface for users to explore the gym and connect with it easily.

## 🚀 Features
- Responsive design (mobile and desktop)
- Modern and clean user interface
- Home section with hero banner
- About section with detailed information
- Services section with descriptions
- Pricing plans (Basic, Standard, Premium)
- Image gallery
- Contact section with form and details
- Smooth scrolling navigation

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💡 Purpose
This project demonstrates the ability to build a real-world business website with proper structure, design, and user experience.
